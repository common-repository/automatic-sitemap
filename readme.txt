== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip the Compressed file.
2. Upload `automatic-sitemap` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in Dashboard.

== Frequently Asked Questions ==

= How to install this plugin? =

Just follow this Steps
1. Unzip the Compressed file.
2. Upload `automatic-sitemap` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in Dashboard.

= How Can I access my sitemap? =

After install the plugin, simply create a new post or page for create sitemap file. Then you will able to access your sitemap in http://yoursitelink.com/sitemap.xml
